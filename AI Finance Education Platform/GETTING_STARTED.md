# Getting Started with WealthNexus AI

Welcome to WealthNexus AI! This guide will help you explore all features of the platform.

## 🚀 First Steps

### 1. Create Your Account

1. Click **"Sign Up"** on the homepage

## 📊 Understanding Your Dashboard

### Key Metrics Explained

**Total Value**
- Current worth of all your holdings + cash
- Updates in real-time
- Includes unrealized gains

**Total Gain**
- Profit/loss since initial investment
- Shown in dollars and percentage
- Green = profit, Red = loss

**Day Change**
- Today's portfolio movement
- Volatile, don't panic on red days!
- Focus on long-term trends

**Cash Balance**
- Available funds for new investments
- Keep some cash for opportunities
- Recommended: 10-15% cash allocation

### Reading Charts

**Line Chart (Performance)**
- Blue line = Your portfolio
- Purple dashed line = Benchmark (market average)
- Above benchmark = outperforming 🎉

**Pie Chart (Allocation)**
- Each color = different stock
- Larger slice = bigger position
- Aim for balanced distribution

## 🆘 Troubleshooting

### Login Issues
- **Problem**: Can't log in
- **Solution**: Any email/password works (demo mode)

### Data Missing
- **Problem**: Lost progress after refresh
- **Solution**: Check if you're using same browser/device

### Slow Performance
- **Problem**: App feels sluggish
- **Solution**: Close unused tabs, restart browser

### Chart Not Loading
- **Problem**: Charts showing empty
- **Solution**: Refresh page, check internet connection

### XP Not Updating
- **Problem**: Completed action but no XP
- **Solution**: XP updates may take 1-2 seconds, refresh if needed
